Rules of Variables:-
1) Variables should not contain keywords:-
int,float,double,for,if

3) Variables should start with A-Z or a-z or _ :-

b=67

tops=100

4) Variables should not start with numbers:-

Conditional Statements:-

1) if statement
2) if else statement
3) else if statement or if..else if..else ladder
4) Nested if statement
5) Switch statement


1) if statement:-
syntax:-

	if(condition)
	{
		True statment;
	}

2) if else statement:-
syntax:-

	if(condition)
	{
		True statement;
	}

	else
	{
		False statement;
	}

3) else if ladder:-
syntax:-

	if(condition)
	{
		statement;
	}

	else if(condition)
	{
		statement;

	}

	else if(condition)
	{
		statement;
	}

	else
	{
		statement;
	}

4) Nested if statement:-
syntax:-

	if(condition)
	{
		if(condition)
		{
			statement;
		}
		else
	}
	else

5) Switch statement or Switch case:-

syntax:-

	switch(expression)
	{
		case 1:
		    statement;
		    break;

		case 2:
		   statement;
		   break;

		case 3:
		 statement;
		 break;

		default:
		    statement;
	}




Loops in C :-

1) For Loop
--> Nested For Loop
2) While Loop
3) Do...While Loop




1) For Loop:-

syntax:-

	for(initialization;condition;updation)
	{
		statement;

	}

--> Nested For Loop:-

syntax:-


	for(initialization;condition;updation)
	{
		for(initialization;condition;updation)
		{
			statement;
		}
	   statement;
	}	

2) While Loop:-

syntax:-

	initialization
	while(condition)
	{
		statement;
		updation;
	}
3) Do...While Loop:-
syntax:-

	initialization
	do
	{
		statement;
		updation;
	}while(condition);	

Jumping statements:-
